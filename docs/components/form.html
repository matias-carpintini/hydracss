# Forms

Examples of building forms with HydraCSS.

<div class="row">
  <div class="col-xs-12">
    <div class="form-group">
      <label for="input" class="label">Label</label>
      <input type="text" placeholder="Placeholder input" class="input" name="input">
    </div>
  </div>
  <div class="col-xs-12 col-md-6">
    <div class="form-group">
      <label for="select" class="label">Label for select</label>
      <select name="select" id="select" class="select">
        <option value="0">Select an option</option>
      </select>
    </div>
  </div>
  <div class="col-xs-12 col-md-6">
    <div class="form-group">
      <label for="select2" class="label">Another label</label>
      <select name="select2" id="select2" class="select">
        <option value="0">Select an option</option>
      </select>
    </div>
  </div>
</div>
